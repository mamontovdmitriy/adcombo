Adcombo, тестовое задание:

Без использования сторонних библиотек сделать свой мини-framework для системы управления контентом сайта без фронтенда
в связке php + mysql. При разработке проекта большим плюсом будет использование namespace.

1. Авторизация по логину и паролю (информация о логине и пароле хранится в базе, пароль в зашифрованном виде).
Все пункты ниже - должны быть доступны только после авторизации.
2. Все страницы системы должны быть ЧПУ. Например: site.ru/pages/1/
3. Вывод на страницу главного меню (Главная, Пользователи, Каталог, Страницы...) с отметкой текущего пункта
4. В разделе «Страницы» сделать возможность добавить, удалить, изменить информацию о страницах.
Информация - название, контент (визуальный редактор), алиас (генерируется транслитом из названия, с проверкой на дубликат).
5. Ссылка «выход из системы».
6. Будет плюсом, если в качестве верстки будет применен bootstrap (версия не важна).
